。本專案旨在研究美國經濟指標與股市走勢關係，範圍在2005.01.19至2022.10.21
。包含指標 : S&P500、CPI、CCI(諮商局)、USDX、U.R.(失業率)、F.R.(聯邦利率)、VIX

。全案模組化編寫，共分成爬蟲部(crawler.py)、資料庫操作部(tosql.py)、資料庫整理部(sqlmodify.py)、出圖部(plot.py)
。以主程式(mainprocess.py)運行，便於除錯
。內附USDX、VIX之json檔

。附贈我對21張圖提出之個人見解(research.txt)
